FirstPass

Image Downloader and Browser

A lightweight, stand-alone image browser and digital imaging workflow accelerator, written by a photographer for photographers. Available exclusively on the Mac App Store.

Core Features

  • Lightweight, stand-alone image browser

  • Digital imaging workflow accelerator

  • Allows automatic download from multiple devices

  • Available on the Mac AppStore

What FirstPass Does

FirstPass provides a practical, no-clutter workflow for importing and organising photos straight off your camera or card. It keeps the interface clean while surfacing the information that matters: exposure data, histograms, ratings, and folder structure.

Key capabilities include:

  • Histogram and image information panel via right-click

  • Bulk file renaming with timestamp and auto-sequence support

  • Favourites folder access from within the browser

  • Export to web with configurable format and size

  • Finder integration for direct file access

  • Multi-file rating support

  • Trackpad gesture navigation

  • Software RAID compatibility
