FREQUENTLY ASKED QUESTIONS

Firstpass

Image Download and Editing Software

 
 

Q1. What is FirstPass?

  1. FirstPass is a Mac application that speeds up “first pass” editing of images. Download, view, sort and delete images using the same intuitive and efficient application. When there are a lot of images to go through, FirstPass makes it extremely easy to simultaneously download the images from multiple cards and quickly sort them using ratings filter.


Q2. Where does FirstPass fit in a digital workflow?

  1. FirstPass is designed to be the first step in a digital workflow. At this point, the user has the maximum number of images to browse and edit. FirstPass is designed to be fast and lightweight to get the user through this step very quickly.


Q3. Does FirstPass replace applications such as Aperture or Lightroom?

  1. No, FirstPass does not replace these applications as it does not do image cataloguing. Instead, FirstPass is designed to complement these applications. First, it allows for an efficient download of the images from multiple sources. And second, it allows the user to quickly view and sort the images, such that unnecessary files can be deleted. FirstPass can also be used as a preliminary step prior to further processing in Adobe Photoshop or CaptureOne.


Q4. How many devices can FirstPass download from simultaneously?

  1. There is no limit on how many devices (memory cards, cameras, etc.) FirstPass can download from simultaneously. For faster download speed, it is recommended that not all card readers are attached to the same USB port.


Q5. How many destinations can FirstPass download to simultaneously?

  1. FirstPass can download to two destinations simultaneously. This enables the user to automatically copy the files to their intended destination, as well as to a backup location (such as an external hard drive).


Q6. Can FirstPass export files?

  1. The current version of FirstPass does not export files in any size or format. If there is demand for this feature, it can be added in future releases.


Q7. Which RAW formats does FirstPass support?

  1. FirstPass uses operating system support for decoding RAW files. This enables FirstPass to support all the RAW formats that are currently supported by Mac OS X. Complete list can be viewed here.


Q8. How is support for new cameras added?

  1. Apple updates RAW camera support very often. As soon as user installs new RAW update from Apple, FirstPass will support it automatically.


Q9. Does FirstPass modify any file content when ratings are changed?

  1. FirstPass does not modify any files. All rating are saved separately in an internal FirstPass database.


Q10. How does the ratings filter work in FirstPass?

  1. The rating filter in FirstPass is available in the main window. By default all the ratings are displayed. By pressing on a specific rating number, the user can filter out the images with that rating (i.e. these images will no longer be shown). This feature helps in narrowing down the files the user needs to look at.  For example, the user can start with a large number of files, and rate the best images as 5 (highest rating), and those that are worst as 1 (lowest rating). To view the best images, filter out all ratings except 5 (by pressing unrated and 1-4 in the ratings filter). The user may also choose the delete all the worst images by viewing only those that have been rated as 1 (by pressing unrated and 2-5 in the ratings filter), selecting them and pressing ‘delete’. Finally, the user can press 5 on the ratings bar to concentrate on all the remaining images.


Q11. How can I get help, report problems or request additional features?
