FirstPass©

Image download and first pass editing software
by LightOnCanvas.com

April 24, 2011
Version : v1.0

Objective

On a good day’s shoot it is not uncommon to shoot thousands of frames spanning multiple memory cards. Downloading these cards normally is very cumbersome as each card has to be downloaded one at a time, even if you have multiple card readers. Secondly, it takes time to go through all these images and do a “first pass” edit of the images (rate, delete, view at 100%, etc.). The objective of this application is to speed up these two tasks considerably, and to do them within the same application. 

Unique features of FirstPass

  1. 1.The first application on Mac to enable image download from multiple sources to multiple destinations.

  2. 2.Works with shared devices to enable download over a network.

  3. 3.Provides a fast and clean interface to do a “first pass” editing of images.

  4. 4.Frees up the user’s time by handling multiple downloads at the same time without requiring the user to swap the cards in and out of a card reader or starting the download manually for each card.

  5. 5.Can be configured to automatically start downloading when a device (or multiple devices) are plugged in.

Quick start guide

There are three main windows in this application:

  1. Browser window

  2. Image window

  3. Download controller window

In addition, a preferences window contains two tabs:

  1. General preferences tab

  2. Download destination and file naming preferences tab

Note: Before starting to use the application it is important that the user review the preferences and set them appropriately.

Main Window

The following is the application’s main window (when there are no devices connected to it). All panes in this window are fully resizable. This window has four parts − toolbar (top), folder browser (left), image browser (right) and additional folder / image viewing controls (bottom).


Below the image browser there are two controls. First is a ratings filter − selecting / pressing on any number will filter out files with that rating. To allow for a quick “first pass” editing, hence this feature can be used to easily work through a large set of images, to view / hide images of a particular rating (or multiple ratings), to delete images of low rating / rejects, etc. The rightmost control is a slider to control the thumbnail size in the image browser.

The left side of the main window displays the folder browser, while the right side acts as an image browser. The folder browser includes a list of mounts i.e. all external drives attached to the system, including network drives as well as direct links to the user’s home folder and the Pictures folder.


The three buttons at the bottom of the folder browser can be used to add a folder, to remove a folder and to refresh the view of the current folder. Double clicking on any folder will load images inside that folder into the image browser on the right.

When there are storage devices or a camera connected to the computer, or the camera button is clicked, the folder list shrinks to accommodate the camera view and thus looks like the image below.



The main toolbar for this window looks like the following image. It can be customized by choosing the leftmost “Customize” button, or completely removed for maximum viewing space.


The functionality provided by these buttons is as follows:

  1. Rename files − rename one or more files by adding prefix or suffix to existing filename, or rename the file completely. When renaming multiple files, removing old filename will rename only the first file and give error for the rest because multiple files cannot have the same name in the same folder.


  1. Cameras − displays or hides the camera list view. A user can pick a camera/device from which to download from.

  2. Download − starts a download session with the currently selected device in the device list, or if none is selected, the first device in the list.

  3. Slideshow − starts a slideshow with currently loaded images in the image browser.

  4. Preferences − displays the preference window, which allows the user to change the download destination and general preferences.


General Preferences

The option “Launch when device is connected” works on a per device basis.  If this option is selected for a particular device, the FirstPass application will launch automatically every time this device is plugged into the system.


When FirstPass application is running, the option “Automatically start download” allows the download to automatically begin for any devices or cards that are plugged into the system.  The related option “Automatically close & eject when done” can only be enabled when the automatic download option is selected.

The sequence number setting is used for sequential naming of files or destination folders. FirstPass updates this number automatically to ensure unique sequence numbering across multiple simultaneous or sequential downloads. The user can also manually reset the sequence number in this preference pane.

Destination Preferences

The destination preference pane allows the user to customize the destination folder(s) as well as the naming convention of the downloaded files. Both the folder and the file names can use a variety of tokens as shown in the above screen capture. A secondary destination provides the option of copying the downloaded images to an additional destination folder. At least one of the primary / secondary destinations must be be provided, and if any specified folder does not already exist, it will be created automatically.



Image Window

The right side of the main window acts as the image browser, which displays thumbnails of the images in the selected folder. Double clicking on any image in the image browser window loads it in a larger image viewer window, as shown below.

This window has three parts − toolbar, image and image icon browser. The toolbar contains multiple items, which are shown and explained below.

Most buttons are self explanatory.  Images can be rotated for viewing (these changes are not saved). Its state does not carry over to the next image.


Each image in this view can be rated by pressing the number keys from 0 to 5 (0 signifies unrated) or by clicking on the appropriate number on the rating panel using a mouse.


Finally, user can move between images using “Left” and “Right” arrow keys or by clicking on the “Previous” and “Next” buttons with a mouse.

When the image is zoomed into such that it is larger than the window size, the user can scroll around the image using the mouse scroll wheel, or using two fingers on the TouchPad.

When “Info” button is pressed, it shows EXIF, IPTC and other information for the displayed image in a window.


Note: To see the image information user must click on the "Details" tab at the top of the information window. This is because of a problem in Mac OSX infra structure. A case has been opened with Apple and this issue will automatically get fixed in newer releases.


When an image is loaded in the image window, a few convenient keyboard shortcuts can be used to speed up the workflow:

    1. -Pressing f displays the image in a full screen mode (pressing f again comes out of full screen mode)

    2. -Pressing z displays the image at 100% (pressing z again fits the image to the screen)

    3. -Pressing i displays the image information


Download Window

One download window is presented for each device / card. The download window has two states − minimized and full size. When download begins, the window reduces to a small size, and displays a progress bar and a cancel button.

When minimized only the bottom part of the window is visible along with a download progress bar.
